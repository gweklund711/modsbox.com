RandErg by Gunnar Eklund

With this web project, users can generate their own Concept 2 Erg workouts based on a few parameters. First of all, there is an option to select an interval length. Shorter intervals contain more sets and short rest time, and longer intervals contain fewer sets and more rest time. From there, the user can generate their workout based off of measurements of either time or distance. So far, I've personally enjoyed using this generator, for it allows me to have greater workout variety when I'm erging at home.
