GT Project 2022: REVERSLE by Gunnar Eklund

The goal of Reversle, my exhibition (pretty much a science fair) project, was to reverse engineer the open source code of Wordle and create new variants of the game. Along with it, a major focus was that exhibition attendees would easily be able to interact with my project by playing it on the website or by scanning the QR codes on my trifold board. I am not claiming that this is my own intellectual property, for I simply modified the existing open source code for the original Wordle website (I obtained the code before the site was bought by The New York Times). Special thanks to the developers and owners of Wordle, for this project wouldn't have been possible without their original code. If you would like to learn more about the design process, either check out my Github or this Google Doc.

FOURDLE
Play the classic game of Wordle with four letter words instead. Each time you load the game a new random word is selected so you can play as much as you want.
SIXLE
For an even harder challenge, play Wordle with six letter words. You get seven guesses now, but it's still tough. You can play this game infinitely as well, just reload the page to get a new word to guess.
STATE-LE
For a unique challenge, try guessing a two letter state abbreviation. It's harder than you think! If you fail, just reload the page to try again.
INFINITE WORDLE
For those who are fans of the classic Wordle game, enjoy the original. Only this time each time you load the page a new word is selected. Play as much as you want, not just once a day.
