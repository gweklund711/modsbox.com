GT Project 2022: REVERSLE by Gunnar Eklund

As a project for an exhibition this year, I decided to learn how to reverse engineer Wordle in order to create my own variants of the game. I am not claiming that this is my own intellectual property, for I simply modified existing open source code.
Special thanks to the developers and owners of the original game, for this project wouldn't have been possible without their original code. If you want to learn more about the design process, either check out my GitHub or this Google Doc.

FOURDLE
Play the classic game of Wordle with four letter words instead. Each time you load the game, a new random word is selected, so you can play as much as you want.
SIXLE
For an even greater challenge, play Wordle with six letter words. You get seven guesses now, but it's still tough. You can play this game infinitely as well; just reload the page to receive a new guessable word.
STATE-LE
For a unique challenge, try guessing a two letter state abbreviation. It's harder than you think! If you fail, just reload the page to try again.
INFINITE WORDLE
For those who are fans of the classic Wordle game, enjoy the original. But with this version, each time you load the page, a new word will be selected. Play as much as you want, not just once daily.
