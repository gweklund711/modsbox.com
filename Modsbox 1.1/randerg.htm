RandErg by Gunnar Eklund

With this web project, users can generate their own Concept 2 Erg workouts based on a few parameters. First of all, there is an option to select a number of sets for the workout. A smaller number of sets results in longer rest times and a larger number of sets results in shorter rest times. After the number of sets is selected, the user can generate their workout based off of measurements of either time or distance. So far, I've personally enjoyed using this generator, for it allows me to have greater workout variety when I'm erging at home.
